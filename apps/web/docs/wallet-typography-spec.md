# Wallet Typography 映射（继承全局规范）

## 1. 说明
- Wallet 页面不单独定义字号体系。
- 统一继承全局规范：`apps/web/docs/typography-spec.md`。
- 本文仅记录 `WalletScreen` 的 token 映射，供实现和 review 快速核对。

## 2. WalletScreen 对照清单
- 页面主标题：`text-2xl font-bold`
- 总资产标签：`text-base`
- 总资产金额：`text-4xl font-bold leading-none`
- 主操作按钮：`text-base font-semibold`
- Today 标题：`text-lg font-bold`
- 日报正文：`text-base`
- 日报辅助信息与链接区：`text-sm`
- Holdings 标题：`text-xl font-bold`
- 资产主信息与价格：`text-base font-semibold`
- 资产副信息与数量：`text-sm`
